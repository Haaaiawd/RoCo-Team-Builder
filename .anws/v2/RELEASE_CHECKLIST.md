## 3. 功能验收检查

### 3.1 US-001: 文字配队
- [ ] 用户发送配队请求，Agent 返回推荐队伍

- [ ] 推荐队伍包含属性、技能、搭配建议

### 3.2 US-002: 截图识别
- [ ] 用户上传截图，Agent 返回识别候选
- [ ] 用户确认拥有列表，界面显示状态提示
- [ ] BYOK 非视觉模型上传截图，前端拦截并提示
- [ ] builtin 轨道额度超限，显示 QUOTA_ 语义提示

### 3.3 US-004: 资料卡片
- [ ] Agent 返回精灵卡片，Rich UI 宿主渲染

- [ ] 精灵卡片包含数据来源跳转

### 3.4 US-006: 白名单路径
- [ ] 终端用户首页不出现禁止入口（Notes, Channels, Knowledge, Admin, Open Terminal）
- [ ] 侧栏导航满足白名单约束

- [ ] 主题注入后，UI 呈现复古手账风

### 3.5 轨道隔离
- [ ] 内置轨道与 BYOK 轨道清晰分离
- [ ] 不会发生静默切轨
- [ ] 切换轨道时有明确的用户提示

### 3.6 视觉风格
- [ ] 炭黑侧栏、撕纸边缘效果正确

- [ ] 暖金高亮、虚线内框正确
- [ ] 用户气泡纸贴片效果正确

- [ ] 胶囊形输入区正确
- [ ] History 项手账标签感正确
